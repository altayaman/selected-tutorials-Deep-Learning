                    Very nice post! Thanks!

                    Some comments on NCE.

                    In original paper of JMLR 2012, the author said their goal is to view Z as a separated parameter independent of other parameter. But this could not be done by using ML on softmax cross entropy. Because it will cause Z-> 0. (when Z=0, the max likelihood is infinity).

                    When Z=0, for any target word, it will get a infinite score(more than 1). So they want to add some un-targrt word information in the loss function to fight againt this. That why the method is called Noise -Contrastive.

                    When changing to the new NCE loss function, we could find when Z=0, the loss function will not be minimized.

                    The goal of origin softmax cross entropy is that:
                    When x is from Data distribute, P(x) should be high. This model will get into Z =0.
                    The goal of NCE is that:
                    The equ 4 in the paper shows that:
                    1.when x is from Data distribute, Q(x) ->0,P(x) should be high
                    2.when x is not from Data distribute, P(x) -> 0.
                    Cause the Z in P(x) is a parameter, P(x) could get to infinity and Z will get to zero. But the second(what noise could help) will make sure Z could not be learned as 0. And the loss function could be optimized well.

                    I think most post or papers make the wrong logic(maybe they want to save space to explain). They all just said NCE define a new loss function and finally this function still need to compute Z.Then NCE use a trick to view Z as a parameter. In actually, to make Z could be optimized is just why NCE loss function is constructed.
