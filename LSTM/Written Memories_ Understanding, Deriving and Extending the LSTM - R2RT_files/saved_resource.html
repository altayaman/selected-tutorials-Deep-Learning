                          r2rt Mod Annon Amous • 2 months ago

                          Consider the 1-dimensional case: Given an input x, we want an affine transformation, ax, followed by a nonlinearity (sigmoid, tanh, relu, etc.) to equal to x, regardless of what x is. Without the nonlinearity, we can set a = 1, and we get the identity, but there is no constant value of a that will make this work when the nonlinearity is there. We could produce a very close to linear function by using a sigmoid/tanh nonlinearity and very small values of a, but then this will be an aggressive contraction (not the identity).

                                              r2rt Mod M.M. • 4 months ago

                                              Hi M.M. The matrix [[1,1],[0,1]] that you give as an example does not have unit singular values: its singular values are 1.618 and 0.618. Also, the start condition isn't We_i = e_i, but rather |We_i| = |e_i|.

                                              A valid complaint, on the other hand, would be that the converse isn't true (unit norm columns do not imply unit singular values). An orthogonal matrix on the other hand, does have unit singular values, and is therefore a commonly used RNN initialization in practice over the Xavior-Glorot initialization. See, e.g., this post: https://smerity.com/article..., and a random paper that takes advantage of orthogonal RNN initializations: https://arxiv.org/pdf/1603..... Perhaps I will update the main post with this information...
